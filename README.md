# MercattoPy

Sistema de gestão desenvolvido em Django para controle de produtos, vendas e análise de dados.

## Funcionalidades

- Cadastro, edição e remoção de produtos  
- Organização de produtos por categorias  
- Registro de vendas e itens de venda  
- Dashboard com métricas gerais (vendas, faturamento, produtos, etc.)  
- Controle de acesso com autenticação de usuários  
- Perfis de usuário com diferentes permissões (Admin e Operador)

## Tecnologias utilizadas

- Python  
- Django  
- SQLite (ambiente de desenvolvimento)  
- HTML / CSS  

## Organização do sistema

O projeto foi estruturado em módulos para facilitar manutenção e expansão, separando funcionalidades como catálogo de produtos e vendas.

## Observações

O acesso às funcionalidades é controlado por autenticação e permissões, garantindo que cada tipo de usuário tenha acesso apenas ao que é permitido.
